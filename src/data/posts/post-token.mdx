# Finalidade
Responsável por realizar validação do Token precadastro. Ao validar um Token, atualiza tabela com data de acesso e altera status de ATIVO para 0.

# Requisição da Atualização da Transação Pagamento Parcelas
`Autenticação: Obrigatório`
| HTTP Verbs | URL                                     | Descrição                                                                                                                         |
|------------|-----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| `GET`      | /token/precadastro/token | Faz validação do token pré cadastro |
## Headers
| Nome          | Valor                           |
|---------------|---------------------------------|
| Authorization | `Bearer {Valor de Token Gerado}` |
| Content-Type  | `application/json`              |

|-----------|-----------|---------------|
|| |


# Resposta da Adição do Log
